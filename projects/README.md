# UtopikCode Docs Index

This repository indexes the documentation for **UtopikCode** projects.

## Documentation collections

### Quickspaces
- **What it is:** the Quickspaces documentation collection
- **Where to read:** [Quickspaces docs landing page](./quickspaces/README.md)
- **Documentation topics:** [Quickspaces docs folder](./quickspaces/docs/)

## Why this structure?

Keeping the docs organized as a project-style index makes it easy to add new documentation collections later while still focusing this repo on UtopikCode documentation.

## Adding future docs

When new docs sets are added, use:

- `projects/<project-name>/README.md`
- `projects/<project-name>/docs/`
- optional `projects/<project-name>/assets/`
